# SchoolScheduleApi

REST API для получения расписания школы. Проект реализован на **Spring Boot** с использованием Maven.

## 📋 Функциональные возможности

- Получение списка всех преподавателей
- Получение списка всех классов
- Получение расписания для указанного класса на конкретную дату (расписание определяется по дню недели)

## 🚀 Запуск проекта

### Предварительные требования
- **JDK 17** или новее
- **Maven** (можно использовать встроенный `mvnw`)
- **Docker** и **Docker Compose** (опционально, для запуска через контейнер)

### Способ 1: Локальный запуск (без Docker)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/iwnttodiexd/ShcoolShceduleApi.git
   cd ShcoolShceduleApi
2. Соберите проект с помощью Maven:
   ```
   ./mvnw clean package или maven clean package
3. Запустите приложение:
   ```
   java -jar target/*.jar
4. Приложение будет доступно по адресу: http://localhost:8080

### Способ 2: Запуск через Docker Compose

1. Убедитесь, что Docker запущен.

2. Выполните в корневой папке проекта:
   ```
   docker-compose up -d
3. Приложение поднимется на http://localhost:8080

 Остановка контейнера:
   ```
   docker-compose down
   ```
